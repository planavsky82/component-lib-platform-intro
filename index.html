@planit Component Library Platform

The platform provides:

  1. a web component library to utilize in your application
  2. a command that allows you to spawn your own web component library

@planit/component-lib-platform will automatically set up a component library development platform in just a few minutes. It includes a Stencil web component starter library, a TailwindCSS configuration, Storybook.js, and a demo application.

Get started! Simply run the following script:

npx @planit/component-lib-platform {your-project-name}
